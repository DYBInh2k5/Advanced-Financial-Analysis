# Sample Data

Thư mục này chứa dữ liệu mẫu để demo dự án.

## Cách sử dụng

1. Chạy script thu thập dữ liệu:
```bash
python scripts/simple-data-collector.py
```

2. Dữ liệu sẽ được lưu vào thư mục `data/` (không được commit lên Git)

## Lưu ý

- Dữ liệu thực tế không được commit lên repository vì lý do bảo mật
- Chỉ commit các file mẫu và hướng dẫn
- API keys phải được lưu trong file local không được commit
